設計思考 2024年10月24日

從數位冷漠到溫暖觸感：尋找介面的靈魂

edit

在這個充斥著螢幕與像素的時代，我們每天滑動過無數光鮮亮麗卻冰冷的介面。我時常在想，數位產品是否注定只能是有效率的工具？它們能否帶有一絲如同實體物件般的溫度與記憶？

回歸物理材質的啟發

翻開一本舊筆記本，那種紙張邊緣的微微磨損、墨水滲透紙纖維的痕跡，都能讓人感到安心。在設計這個作品集時，我試圖將這種「觸覺」帶入視覺之中。不再依賴強烈的陰影與漂浮感，而是運用如同紙張般的層次與柔和的邊緣。

"The best interfaces are those that don't shout 'technology', but whisper 'human'."
— A note to myself

色彩作為情緒的載體

選用奶油白與大地棕作為主色調，並非偶然。這是一種拒絕過度刺激的宣言。它邀請讀者放慢腳步，像閱讀一封手寫信一樣，慢慢咀嚼每一個字。我們使用微小的圓角與不對稱的排版，打破網格帶來的僵硬感，讓畫面多了一點「有機」的呼吸空間。

最終，設計不僅僅是為了解決問題，更是為了創造連結。當一個介面能讓人感到寧靜與專注時，也許我們就成功地在數位世界裡，留下了一點人性的溫暖。
